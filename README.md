# OpenHands MCP GitHub Agent

An autonomous AI agent built using OpenHands SDK and MCP (Model Context Protocol).

## Features

- File system operations
- GitHub repository access
- Create/update files
- Create branches
- Open pull requests
- Commit code automatically
- MCP server integration

## Technologies Used

- Python
- OpenHands SDK
- MCP
- GitHub MCP Server
- OpenAI GPT-4o-mini
